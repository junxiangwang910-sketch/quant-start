# 04 Strategy Development

This project focuses on turning previous research and backtesting results into a more complete strategy prototype.

## Objective

- Define a clear trading strategy
- Specify entry and exit rules
- Add simple risk control ideas
- Evaluate strategy performance more completely

## Strategy Prototype

This project will build a simple momentum rotation strategy.

Core idea:

- rank assets by momentum
- hold the strongest asset
- rebalance on a fixed schedule
- compare performance with a benchmark
- add simple risk control rules
