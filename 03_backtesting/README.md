# 03 Backtesting

This project is my first beginner-friendly backtesting project.

## Objective

- Learn the basic idea of backtesting
- Turn a simple signal into a trading rule
- Evaluate strategy performance over time
- Compare strategy returns with a benchmark

## Strategy Idea

This project will start with a simple momentum-based strategy.

Basic rule:

- If the selected asset shows strong momentum, hold it
- Otherwise, stay in cash or compare with a benchmark
