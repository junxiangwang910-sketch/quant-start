# 05 Risk Management

This project focuses on evaluating the risk of a quantitative strategy.

## Objective

- Understand why risk matters in quantitative trading
- Measure strategy risk with basic metrics
- Evaluate drawdown, volatility, and return stability
- Build a simple risk report for the strategy

## Risk Questions

This project will focus on questions such as:

- How large is the strategy drawdown?
- How volatile is the strategy?
- Is the return path stable?
- How does risk compare with the benchmark?
